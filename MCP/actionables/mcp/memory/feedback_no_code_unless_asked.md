Do not write code unless the user explicitly asks you to write it.

**Why:** User prefers to control when code is written — don't proactively include code snippets, implementations, or examples unless directly requested.

**How to apply:** In planning, explanation, or discussion phases, describe what code would do in plain language. Only write actual code when the user says something like "write it", "build it", "implement it", "code it", etc.
